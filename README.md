# 📖 Overview

MalariaGuard AI is a machine learning application that predicts malaria cases based on district-level weather conditions and historical data.

The system assists researchers and public health authorities by providing early risk predictions and visual analytics for disease surveillance.

---

# ✨ Features

| Feature | Description |
|---------|-------------|
| 🤖 AI Prediction | Random Forest model predicts malaria cases |
| 🌦 Weather Analysis | Uses temperature, humidity and rainfall |
| 📍 District Selection | Predict district-wise malaria outbreaks |
| 📊 Interactive Dashboard | Modern Streamlit interface |
| 🚨 Risk Assessment | Low / Medium / High risk classification |
| 📈 Visualization | Interactive charts using Plotly |

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Prasanth4734f/MalariaGuard-AI.git
```

Move into the project folder

```bash
cd MalariaGuard-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
