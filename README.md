# 🦟 About MalariaGuard AI

MalariaGuard AI is an **Artificial Intelligence-powered disease prediction platform** designed to estimate malaria risk using environmental and historical health data.

The application combines **Machine Learning**, **interactive visualization**, and a **modern Streamlit interface** to assist researchers, students, and healthcare professionals in analyzing malaria trends and predicting disease outbreaks.

---

# ✨ Key Features

| Feature | Description |
|----------|-------------|
| 🤖 AI Prediction | Predict malaria cases using Random Forest |
| 🌦 Weather Analysis | Temperature, Humidity & Rainfall |
| 📍 District-wise Prediction | Predict malaria risk for each district |
| 📊 Interactive Dashboard | Modern Streamlit user interface |
| 🚨 Risk Classification | Low, Medium & High Risk Detection |
| 📈 Visualization | Interactive charts powered by Plotly |
| ⚡ Fast Prediction | Instant ML inference |
| 🔍 Clean Interface | User-friendly navigation |

# 🚀 Installation Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Prasanth4734f/MalariaGuard-AI.git
```

---

### 2️⃣ Navigate into the Project

```bash
cd MalariaGuard-AI
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Launch the Application

```bash
streamlit run app.py
```

# 💻 Using the Application

1. Open the Home Page
2. Click **Start Prediction**
3. Select a District
4. Click **Predict**
5. View the prediction dashboard
6. Analyze:
   - Predicted Cases
   - Weather Conditions
   - Risk Level
   - Visual Charts
